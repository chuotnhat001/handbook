Competency & Checklist

Checklist hoàn thành từng phase, competency roadmap và milestone học tập

Sử dụng checklists để đánh giá tiến độ học tập và sẵn sàng cho từng phase triển khai
Cách sử dụng Checklist Mỗi checklist item đại diện một năng lực có thể chứng minh được (demonstrable competency). Hoàn thành = có thể thực hiện independently, không chỉ là đã đọc tài liệu. Gate review nên có peer hoặc mentor xác nhận.

Phase 1: Foundation — Checklist hoàn thành

  • Trình bày được CRM strategy và 5 loại CRM (Operational, Analytical, Collaborative, Social, AI-driven)
  • Vẽ được Customer Journey Map hoàn chỉnh với touchpoints, emotions, pain points
  • Xây dựng Business Case với ROI calculation và TCO analysis
  • Đánh giá được CRM vendors sử dụng scorecard chuẩn hóa (minimum 3 vendors)
  • Viết được SRS document theo IEEE 830 với user stories và acceptance criteria
  • Mô hình hóa được 2+ quy trình BPMN 2.0 (AS-IS và TO-BE)
  • Tạo được Data Dictionary với entity definitions và relationships
  • Xây dựng Project Charter với scope, timeline, RACI, và risk register
  • Thiết kế Communication Plan cho stakeholders (min 4 stakeholder groups)
  • Tạo Change Management Plan dựa trên ADKAR model

Phase 2: Domain Knowledge — Checklist hoàn thành

Sales Domain

  • Thiết kế được Sales Process với stage gates và qualification criteria (MEDDPICC)
  • Xây dựng Pipeline Forecasting Model với weighted probability
  • Định nghĩa được CPQ rules cho ít nhất 1 product line

Marketing Domain

  • Thiết kế được Lead Scoring model (behavior + demographic)
  • Xây dựng multi-touch campaign journey (min 5 steps, 2 channels)
  • Định nghĩa MQL → SQL handoff criteria và SLA

Service Domain

  • Thiết kế Case Lifecycle với SLA tiers và escalation matrix
  • Xây dựng Knowledge Base taxonomy (min 3 levels deep)
  • Định nghĩa Omnichannel routing rules (min 3 channels)

Data & Analytics Domain

  • Thiết kế Customer 360 data model (unified profile schema)
  • Xây dựng KPI Dashboard specification (min 10 KPIs across domains)
  • Tạo Data Quality scorecard với measurement dimensions

Phase 3: Architecture & Design — Checklist hoàn thành

  • Tạo Architecture Vision Document theo TOGAF ADM
  • Xây dựng Business Capability Map (L0-L2) với heat map analysis
  • Thiết kế Context Map (DDD) với ít nhất 4 bounded contexts
  • Viết Ubiquitous Language glossary cho mỗi bounded context
  • Thiết kế RESTful API (min 10 endpoints) với OpenAPI 3.1 spec
  • Tạo C4 Diagrams đầy đủ 4 levels cho 1 service
  • Xây dựng Data Model (conceptual → logical → physical) cho core entities
  • Thiết kế Integration Architecture với event catalog
  • Hoàn thành Security Architecture: IAM design, RBAC model, threat model (STRIDE)
  • Viết ít nhất 5 Architecture Decision Records (ADRs)
  • Hoàn thành DPIA (Data Protection Impact Assessment)

Phase 4: Implementation — Checklist hoàn thành

  • Implement ít nhất 2 bounded contexts theo Clean Architecture
  • Đạt test coverage >80% (unit + integration)
  • Xây dựng CI/CD pipeline hoàn chỉnh (build → test → scan → deploy)
  • Implement Workflow Engine với ít nhất 3 workflow types
  • Xây dựng Extension Framework với plugin registration
  • Hoàn thành Performance Test với documented baselines
  • Execute Data Migration thành công (source → target với validation)
  • Zero-downtime deployment chứng minh được (blue-green hoặc canary)
  • Security scan clean (no critical/high vulnerabilities)
  • Frontend component library với Storybook documentation

Phase 5: Operations — Checklist hoàn thành

  • Viết Operational Runbook cho top 10 common tasks
  • Thiết lập SLO Dashboard với SLI measurement (min 4 services)
  • Implement alerting strategy với noise ratio < 5%
  • Hoàn thành Incident Response drill (simulate + resolve + postmortem)
  • Đạt 99.9% uptime trong 30 ngày liên tục
  • Chứng minh auto-scaling hoạt động đúng (load test → scale up → scale down)
  • Pass security audit (internal hoặc external penetration test)
  • Thiết lập Adoption Dashboard với min 5 engagement metrics
  • Conduct Quarterly Business Review (QBR) với executive stakeholders

Phase 6: Advanced — Checklist hoàn thành

  • Deploy ít nhất 1 ML model to production (lead scoring hoặc churn prediction)
  • Implement RAG-based AI assistant cho CRM knowledge retrieval
  • Xây dựng multi-tenant isolation (prove tenant data cannot leak)
  • Launch Developer Portal với SDK và sandbox environment
  • Build Low-code form/workflow builder hoạt động end-to-end
  • Demonstrate AI safety guardrails (prompt injection protection, output validation)
