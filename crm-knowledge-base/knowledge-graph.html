Learning Path — Lộ trình học tập

Path A: Full Sequential (đề xuất cho beginners)

Thời gian: ~640 giờ (16 tuần full-time hoặc 32 tuần part-time)

  1. Phase 1 tuần tự (80h) → Gate: Business Case + Requirements Document
  2. Phase 2 có thể parallel 4 domains (120h) → Gate: Process Diagrams + KPI Framework
  3. Phase 3 tuần tự (120h) → Gate: Architecture Decision Records + C4 Diagrams
  4. Phase 4 tuần tự (140h) → Gate: Working Prototype + Test Suite
  5. Phase 5 tuần tự (100h) → Gate: Runbooks + SLO Dashboard
  6. Phase 6 chọn lọc (80h) → Gate: Innovation Proof of Concept

Path B: Role-Based (cho chuyên gia chuyên ngành)

Path C: Fast Track (minimum viable knowledge)

DOC-101 → DOC-105 → DOC-108 → DOC-303 → DOC-401 → DOC-408

~60 giờ — minimum để hiểu end-to-end flow, không đủ sâu cho production implementation.
