Danh Mục Sơ Đồ Cần Xây Dựng

BPMN 2.0 — Business Process Diagrams

Sơ đồPhaseMô tả
Lead-to-Cash ProcessP2End-to-end từ lead capture đến revenue recognition
Case Resolution ProcessP2Từ case creation đến resolution + satisfaction survey
Campaign Execution ProcessP2Planning → Execution → Measurement → Optimization
Approval WorkflowP4Generic approval engine (discount, contract, refund)
Onboarding ProcessP5New customer/user onboarding end-to-end

UML 2.x — System Design Diagrams

Sơ đồPhaseMô tả

State Machine DiagramsP3Lead states, Case states, Opportunity stages
Activity DiagramsP2Complex business logic flows
Component DiagramP3Service boundaries and dependencies
Deployment DiagramP4Infrastructure topology per environment

C4 Model — Architecture Documentation

LevelPhaseMô tả

L2: ContainerP3Applications, databases, message brokers
L3: ComponentP3-P4Internal structure of each container
L4: CodeP4Class-level design for complex modules
