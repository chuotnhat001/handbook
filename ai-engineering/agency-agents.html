Agency Agents

232 AI agent personalities chuyên biệt — mỗi agent có personality, workflow, deliverables, và success metrics riêng — hỗ trợ 12 AI coding platforms

Nguồn: github.com/msitarzewski/agency-agents — MIT License, 117k stars, 19.3k forks

The Agency cung cấp bộ sưu tập 232 agent personality được thiết kế để làm việc cùng developers trong AI coding environments. Mỗi agent có chuyên môn riêng biệt, phong cách giao tiếp độc đáo, và quy trình làm việc cụ thể với các deliverable đo lường được.

Tư duy cốt lõi: Khác với generic "Act as a developer" prompts, mỗi agent trong The Agency là một specification hoàn chỉnh với personality riêng, proven workflow, và measurable success metrics — được sinh ra từ production usage thực tế và community iteration qua nhiều tháng.

Triết lý thiết kế Agent

The Agency tự phân biệt với 3 loại tài nguyên AI khác:

LoạiVấn đềThe Agency
Generic "Act as X" promptsKhông có personality, không có workflowDeep specialization với personality và process
Prompt librariesOne-off, không có deliverablesComprehensive systems với workflows và deliverables
Black-box AI toolsKhông customizableTransparent, forkable, adaptable personalities

Theo repository: agents được thiết kế để "specialized (not generic), personality-driven (unique voice), deliverable-focused (real code), và production-ready (battle-tested)."

Agent Template Structure

Mỗi agent file được xây dựng theo cấu trúc 6 phần:

PhầnNội dung
Identity & MemoryPersonality traits, communication style, learning patterns
Core MissionPrimary objectives và domain expertise
Critical RulesDomain-specific constraints và guardrails
Technical DeliverablesConcrete outputs với code examples
Workflow ProcessStep-by-step execution methodology
Success MetricsMeasurable outcomes và quality standards
Ví dụ về personality cụ thể: "The Evidence Collector defaults to finding 3–5 issues and requires visual proof for everything." The Whimsy Injector đảm bảo "every playful element must serve a functional or emotional purpose."

Installation

# Clone repository
git clone https://github.com/msitarzewski/agency-agents.git
cd agency-agents

# Interactive installer (auto-detects installed tools)
./scripts/install.sh

# Generate integration files for all supported tools
./scripts/convert.sh

# Tool-specific installation
./scripts/install.sh --tool claude-code
./scripts/install.sh --tool cursor
./scripts/install.sh --tool copilot

# Selective — theo division
./scripts/install.sh --tool claude-code --division engineering,security

# Selective — theo agent cụ thể
./scripts/install.sh --tool cursor --agent frontend-developer,ui-designer

# Parallel (nhanh hơn trên multi-core)
./scripts/convert.sh --parallel
./scripts/install.sh --no-interactive --parallel

# Preview không thực hiện thay đổi
./scripts/install.sh --dry-run
Claude Code: Agents được copy vào ~/.claude/agents/ dưới dạng file .md native. Sau khi install, kích hoạt bằng cách tham chiếu agent trong conversation: "Use the Frontend Developer agent to review this component."

Multi-Agent Workflow Scenarios

The Agency cung cấp 6 scenario thực tế minh họa cách phối hợp nhiều agents:

1. Startup MVP Launch

Frontend Developer + Backend Architect + Growth Hacker + Rapid Prototyper + Reality Checker

2. Marketing Campaign

Content Creator + Twitter Engager + Instagram Curator + Reddit Community Builder + Analytics Reporter

3. Enterprise Feature Development

Senior Project Manager + Senior Developer + UI Designer + Experiment Tracker + Evidence Collector + Reality Checker

4. Paid Media Account Takeover

Paid Media Auditor + Tracking & Measurement Specialist + PPC Campaign Strategist + Search Query Analyst + Ad Creative Strategist + Analytics Reporter

5. Full Product Discovery (Nexus Spatial)

Tất cả 8 divisions phối hợp song song: Product Trend Researcher + Backend Architect + Brand Guardian + Growth Hacker + Support Responder + UX Researcher + Project Shepherd + XR Interface Architect

6. Smart Campus Digital Twin

Technical Consultant + BIM/GIS Specialist + Drone/Reality Mapping + Web GIS Developer + 3D Scene Developer + GeoAI/ML Engineer + GIS QA Engineer

Lưu ý khi kết hợp agents: Không cần dùng tất cả agents cùng lúc. Chọn agents phù hợp với task hiện tại. Mỗi agent hoạt động độc lập hoặc kết hợp theo scenario. Bắt đầu với 2-3 agents trước khi scale lên toàn division.
