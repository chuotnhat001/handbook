GStack

Bộ 30+ slash commands và skills cho AI-powered development — từ planning, design, code review đến deploy và monitoring — tích hợp 10+ AI coding agents

Nguồn: github.com/garrytan/gstack — by Garry Tan (YC President), MIT License, 118k stars

GStack là bộ AI development skills mã nguồn mở do Garry Tan (President của Y Combinator) tạo ra. Nó cung cấp 30+ slash commands bao phủ toàn bộ vòng đời phát triển phần mềm: từ ideation, planning, design, coding, review, testing, cho đến ship và monitoring. GStack hoạt động với Claude Code, OpenAI Codex, Cursor, Kiro, và nhiều AI agents khác.

Tư duy cốt lõi: GStack không chỉ là một tập lệnh — đây là một workflow hoàn chỉnh theo triết lý "Think → Plan → Build → Review → Test → Ship → Reflect", nơi mỗi skill feed vào skill tiếp theo, không có gì bị bỏ sót. Garry Tan báo cáo ~810× tăng năng suất so với 2013.

Cài Đặt

Yêu cầu

  • Claude Code (hoặc AI agent được hỗ trợ)
  • Git
  • Bun v1.0+
  • Node.js (chỉ cần trên Windows)

Cài đặt nhanh (30 giây)

git clone --single-branch --depth 1 https://github.com/garrytan/gstack.git ~/.claude/skills/gstack
cd ~/.claude/skills/gstack
./setup

Sau đó thêm phần gstack vào CLAUDE.md của project:

## gstack
Use /browse from gstack for all web browsing.
Available skills: [full list from README]

Team Mode (tự động cập nhật cho shared repos)

(cd ~/.claude/skills/gstack && ./setup --team)
~/.claude/skills/gstack/bin/gstack-team-init required
git add .claude/ CLAUDE.md
git commit -m "require gstack for AI-assisted work"

Sprint Workflow

GStack được thiết kế theo triết lý sprint hoàn chỉnh, mỗi skill feed vào skill tiếp theo:

Think→ Plan→ Build→ Review→ Test→ Ship→ Reflect

Các skills tích hợp với nhau theo chuỗi:

  • /office-hours generate design docs mà /plan-ceo-review đọc
  • /plan-eng-review viết test plans mà /qa sẽ pick up
  • /review auto-fix issues và flag race conditions để người dùng quyết định
  • /ship audit coverage và generate regression tests từ /qa fixes
Voice Input: GStack respond với natural language — "run a security check", "test the website", "do an engineering review" — đều trigger đúng skill tương ứng mà không cần nhớ tên lệnh chính xác.

GBrain: Persistent Knowledge Base

GBrain là tính năng optional cho phép agent giữ context và knowledge giữa các sessions.

/setup-gbrain

Có 4 cách thiết lập:

  1. Supabase existing URL — paste Session Pooler URL
  2. Supabase auto-provision — paste Personal Access Token
  3. PGLite local — zero accounts, 30 giây setup
  4. Remote gbrain MCP — qua Tailscale/ngrok/internal LAN

Sync Commands

/sync-gbrain                    # Incremental (default)
/sync-gbrain --full             # Full reindex
/sync-gbrain --dry-run          # Preview changes trước khi sync

GBrain hỗ trợ per-repo trust tiers: read-write, read-only, deny — giúp kiểm soát quyền truy cập knowledge theo từng repository.

Parallel Sprint Management

GStack tích hợp với Conductor để chạy 10-15 sprints song song:

  • Mỗi sprint chạy trong isolated workspace
  • Unified sprint tracking across all branches
  • Smart review routing — CEO không review infra fixes
  • Test coverage audits tự động trên mỗi /ship
  • Automatic regression test generation từ /qa fixes

Browse & Chrome DevTools Protocol

/browse và /open-gstack-browser cho phép agent điều khiển real Chromium browser.

Domain Skills

$B domain-skill save    # Lưu per-site notes (e.g., "LinkedIn Apply is in iframe")
                        # Notes tự động kích hoạt khi quay lại site đó

Raw CDP (Chrome DevTools Protocol)

$B cdp <Domain.method>  # Raw CDP escape hatch
                        # Deny-default, cần allowlist trước khi dùng
Lưu ý CDP: Raw Chrome DevTools Protocol commands bị deny theo mặc định. Cần thiết lập allowlist rõ ràng trước khi dùng $B cdp.

Ví Dụ End-to-End

Ví dụ build "daily briefing app for calendar" từ ý tưởng đến PR:

You:    I want to build a daily briefing app for my calendar.
        /office-hours

Claude: [asks about pain points, challenges framing, generates 3 approaches]
        RECOMMENDATION: Ship the narrowest wedge tomorrow.

You:    /plan-ceo-review
        [reads design doc, challenges scope, runs 10-section review]

You:    /plan-eng-review
        [ASCII diagrams, test matrix, security concerns]

You:    Approve plan.
        [writes 2,400 lines across 11 files in ~8 minutes]

You:    /review
        [AUTO-FIXED 2 issues, flags race condition for your approval]

You:    /qa https://staging.myapp.com
        [real browser test, finds and fixes bug]

You:    /ship
        Tests: 42 → 51 (+9 new). PR: github.com/you/app/pull/42

Gỡ Cài Đặt

Khi vẫn còn repo

~/.claude/skills/gstack/bin/gstack-uninstall

Khi không có repo (manual)

pkill -f "gstack.*browse" 2>/dev/null || true
rm -rf ~/.claude/skills/gstack ~/.gstack
rm -rf ~/.codex/skills/gstack* ~/.factory/skills/gstack*
rm -f /tmp/gstack-*

Sau đó xóa phần ## gstack khỏi CLAUDE.md của mỗi project.
