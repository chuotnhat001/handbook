Obsidian

Ứng dụng ghi chú và quản lý kiến thức cá nhân. Miễn phí, local-first, dùng Markdown thuần.

Ý tưởng cốt lõi: Ghi chú của bạn thuộc về bạn. Obsidian lưu trữ mọi thứ dưới dạng file Markdown trên máy local, không phụ thuộc vào cloud hay vendor nào. Bạn có thể đọc notes bằng bất kỳ text editor nào, mãi mãi.

Triết lý thiết kế

Obsidian xây dựng trên 3 nguyên tắc:

  • Privacy: Notes lưu trên thiết bị của bạn. Obsidian không thu thập telemetry, không bán dữ liệu người dùng.
  • Flexibility: Hàng nghìn plugins và themes tùy chỉnh workflow theo ý bạn.
  • Longevity: File format mở (Markdown). Dữ liệu của bạn accessible mãi mãi, không bị lock-in.

Nền tảng hỗ trợ

  • Windows, macOS, Linux (desktop)
  • iOS, Android (mobile)
  • Web Clipper (browser extension)
  • CLI tool

Linking & Graph View

Tạo kết nối giữa các notes bằng cú pháp [[tên note]]. Mỗi link tạo ra một backlink tự động ở note đích. Theo thời gian, bạn xây dựng được một mạng lưới kiến thức cá nhân (personal wiki).

Graph View

Hiển thị trực quan tất cả notes và connections dưới dạng đồ thị. Giúp phát hiện patterns và relationships mà bạn không nhận ra khi đọc từng note riêng lẻ.

Backlinks

Mỗi note tự động hiển thị danh sách tất cả notes khác đang link đến nó. Không cần tạo link hai chiều thủ công.

Canvas

Không gian vô hạn để brainstorm, vẽ diagram, và sắp xếp ý tưởng tự do. Kéo thả notes, hình ảnh, và cards trên một canvas không giới hạn.

Plugins & Tùy chỉnh

Obsidian có open API hỗ trợ hàng nghìn community plugins. Bạn có thể biến Obsidian thành bất kỳ công cụ nào bạn cần.

Themes

Hàng trăm themes cộng đồng thay đổi giao diện hoàn toàn. Có thể tùy chỉnh CSS riêng.

Obsidian Sync

Dịch vụ đồng bộ chính thức, end-to-end encrypted (AES-256).

  • Đồng bộ cross-device (desktop + mobile)
  • End-to-end encryption (Obsidian không thể đọc dữ liệu của bạn)
  • Version history 1 năm
  • Shared vaults cho team collaboration
  • Selective sync (chọn folders cần đồng bộ)
Alternatives miễn phí: Bạn không bắt buộc dùng Obsidian Sync. Vì notes là file Markdown thường, bạn có thể đồng bộ bằng iCloud, Google Drive, Dropbox, Syncthing, hoặc Git.

Obsidian Publish

Biến notes thành website công khai: wiki, knowledge base, hoặc digital garden.

  • Custom domain
  • Themes tùy chỉnh
  • Graph view trên web
  • Full-text search
  • Password protection
  • SEO optimized

Bắt đầu nhanh

  1. Download: Tải từ obsidian.md (miễn phí)
  2. Tạo Vault: Chọn một folder trên máy làm vault (nơi chứa tất cả notes)
  3. Viết note đầu tiên: Tạo file mới, viết bằng Markdown
  4. Link notes: Dùng [[tên note]] để tạo kết nối giữa các notes
  5. Khám phá Graph: Mở Graph View để xem mạng lưới kiến thức

Workflow cơ bản

Daily Notes

Mỗi ngày tạo 1 note mới (tự động bằng plugin Daily Notes). Ghi lại suy nghĩ, tasks, meetings. Link đến các notes liên quan.

Zettelkasten Method

Viết notes ngắn, mỗi note 1 ý tưởng. Link chúng lại với nhau. Theo thời gian, mạng lưới kiến thức tự hình thành.

Project-based

Tạo folder cho mỗi project. Dùng MOC (Map of Content) note làm index cho project đó.

Tips cho người mới

  • Đừng over-organize: Bắt đầu viết trước, tổ chức sau. Links quan trọng hơn folders.
  • Dùng templates: Tạo template cho meeting notes, daily notes, project notes để tiết kiệm thời gian.
  • Backlinks > Folders: Thay vì phân loại note vào folder, link chúng lại. Graph View sẽ tự tổ chức.
  • Bắt đầu nhỏ: Cài 2-3 plugins thôi. Thêm dần khi cần.
  • Viết cho mình: Notes không cần hoàn hảo. Viết nhanh, link nhiều, refine sau.
