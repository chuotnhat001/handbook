Skills là gì

Skills là shortcut cho context — đóng gói toàn bộ quy trình + bối cảnh vào một lệnh /command. Thay vì prompt đi prompt lại, bạn chỉ cần gõ /tên-skill.

Cách tạo Skill đầu tiên

Cách 1: Dùng Skill Creator có sẵn

Mở Cowork → Gõ yêu cầu → Trả lời phỏng vấn → Skill-creator sinh SKILL.md → Lưu & cài đặt

Cách 2: makemyskill.com (miễn phí)

  1. Vào trang makemyskill.com
  2. Mô tả skill muốn tạo
  3. Tải skill về rồi upload lên Claude

Truy cập kho Skills có sẵn

Đường dẫn: Customize → Personal plugins → Browse plugins

7 mẹo nâng cao về Skills

  1. Kiểm tra Claude hiểu đúng: Hỏi "When would you use the [tên]-skill?"
  2. Negative triggers quan trọng hơn positive: Định nghĩa rõ khi nào KHÔNG dùng skill
  3. Tách biệt: Skill lo quy trình, file giọng văn lo văn phong
  4. Xây từ chat history: Khai thác lại những cuộc trò chuyện cũ đã hiệu quả
  5. Tối ưu token: Task từng cần 15 lượt (12.000 token) → chỉ còn 2 câu hỏi (6.000 token)
  6. Thêm vào prompt: "Take your time. Quality over speed."
  7. Skills portable: Dùng được trên nhiều platform khác nhau
