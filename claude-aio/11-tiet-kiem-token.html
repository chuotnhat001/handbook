Dùng Claude Tiết Kiệm Token Và Hiệu Quả Hơn

23 tips giảm token khi dùng Claude — sắp xếp từ ít ai biết đến phổ biến nhất

Nguồn: phandonggiang.com — Cấp độ 3, Nâng cao
Tư duy cốt lõi: Token = tiền. Mỗi từ bạn gõ và mỗi file bạn upload đều tiêu tốn token. Tiết kiệm token không chỉ giảm chi phí mà còn giúp Claude hoạt động chính xác hơn vì context window không bị quá tải.

Chuyển đổi định dạng file trước khi upload

File PDF và Word chứa nhiều metadata ẩn tiêu tốn token không cần thiết. Chuyển sang plain text hoặc markdown trước khi upload giúp giảm đáng kể lượng token tiêu thụ.

Mẹo: Copy nội dung từ file → paste vào notepad → save as .txt → upload. Giảm 30-50% token so với upload PDF trực tiếp.

Lập kế hoạch trong Chat trước khi làm trong Cowork

Cowork tiêu tốn token nhiều hơn Chat vì nó đọc toàn bộ thư mục. Thảo luận kế hoạch trong Chat trước, sau đó mới chuyển sang Cowork khi đã rõ ràng.

Dùng tính năng hỏi ngược của Claude

Thêm "Hãy hỏi tôi trước" vào prompt. Claude sẽ hỏi làm rõ thay vì đoán → ít lần thử lại → tiết kiệm token.

Batch tasks — gộp nhiều việc nhỏ

Thay vì gửi 5 tin nhắn riêng cho 5 email cần viết, gộp tất cả vào 1 prompt. Giảm overhead từ mỗi lượt gửi/nhận.

Edit message thay vì reply mới

Khi Claude trả lời chưa đúng ý, edit tin nhắn trước đó thay vì gửi tin mới. Tin nhắn mới thêm toàn bộ context cũ vào token count, edit thì không.

Quan trọng: Mỗi reply mới = Claude đọc lại TOÀN BỘ cuộc trò chuyện từ đầu. Cuộc chat dài 50 tin nhắn = mỗi reply mới tốn gấp 50 lần tin nhắn đầu tiên.

Restart conversation khi đã đủ dài

Sau 15-20 tin nhắn, chất lượng giảm và token tăng vọt. Mở chat mới, dán lại thông tin cần thiết — rẻ hơn nhiều so với tiếp tục chat cũ.

Dùng Projects cho file lặp lại

Nếu bạn upload cùng một file trong nhiều cuộc trò chuyện, đưa file đó vào Project. Claude sẽ đọc file từ Project mà không tính vào token hội thoại.

Token tiết kiệm = thêm lượt sử dụng mỗi ngày. Một task từng cần 15 lượt qua lại (12.000 token) có thể giảm xuống 2 câu hỏi (6.000 token) nếu cung cấp context đúng cách.
