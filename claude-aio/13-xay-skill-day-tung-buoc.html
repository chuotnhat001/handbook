Xây Claude Skill Dạy Bạn Từng Bước

Tạo skill /how-to — biến Claude thành hướng dẫn viên cá nhân cho mọi task

Nguồn: phandonggiang.com — Cấp độ 3, Nâng cao
Tư duy cốt lõi: Giá trị cốt lõi không nằm ở skill mà ở khả năng giải quyết vấn đề sâu hơn — "cái nút không bấm được, khoảnh khắc mọi thứ vỡ ra".

Skill /how-to là gì

Skill /how-to hoạt động như bộ hướng dẫn từng bước: Claude đặt câu hỏi làm rõ về task bạn muốn làm, sau đó tạo kế hoạch có cấu trúc để bạn follow.

Gõ /how-to → Mô tả task → Claude hỏi làm rõ → Kế hoạch từng bước

Cách cài đặt và sử dụng

  1. Tải file skill về (hoặc tạo bằng Skill Creator)
  2. Upload lên Claude: Customize → Skills → Create skill
  3. Kích hoạt: gõ /how-to trong chat mới
  4. Mô tả task muốn làm — Claude sẽ phỏng vấn và đưa hướng dẫn

Gợi ý sử dụng

  • Dùng Claude Cowork với Opus 4.8
  • Chế độ effort: High
  • Skill hoạt động tốt nhất khi bạn mô tả rõ mục tiêu cuối cùng
Ví dụ: Gõ /how-to rồi nói "Tôi muốn setup CI/CD cho repo Next.js trên Vercel" — Claude sẽ hỏi về framework version, test coverage, deploy environment... rồi đưa plan chi tiết.

Triết lý đằng sau

Giá trị cốt lõi không nằm ở skill mà ở "cái nút không bấm được, khoảnh khắc mọi thứ vỡ ra" — khả năng giải quyết vấn đề sâu hơn ngoài những hướng dẫn step-by-step thông thường.

Skill chỉ là container. Điều quan trọng là quá trình Claude phỏng vấn bạn — đó là lúc bạn buộc phải suy nghĩ rõ ràng về mục tiêu, constraints, và context của task.
