Ngừng Prompt Theo Kiểu Cũ

Phương pháp prompt hiện đại với Obsidian + Cowork — vượt qua template cứng nhắc

Nguồn: phandonggiang.com — Cấp độ 3, Nâng cao
Tư duy cốt lõi: Prompt ngắn cụt cho kết quả tầm tầm vì thiếu ngữ cảnh. Giải pháp không phải viết prompt dài hơn — mà là xây hệ thống file context + Cowork để Claude tự đọc mọi thứ nó cần.

Các vấn đề với cách prompt cũ

  • Prompt ngắn cụt → kết quả tầm tầm, chung chung
  • Thiếu ngữ cảnh: phong cách viết, đối tượng đọc, context riêng
  • Projects của Claude cải thiện nhưng vẫn cần thiết lập lặp lại
  • Quản lý file .md phức tạp nếu không có tool hỗ trợ

Claude Cowork là gì

  • Kết nối trực tiếp với thư mục máy tính
  • Tự động đọc file văn bản trước mỗi phiên làm việc
  • Có thể tạo file thực tế (docx, ppt, Excel)
Vấn đề: Quản lý file .md phức tạp nếu không có tool hỗ trợ — cần giải pháp giao diện để edit và tổ chức dễ dàng.

Obsidian là giải pháp

Một ứng dụng miễn phí, mở một thư mục file và biến nó thành cơ sở kiến thức gọn gàng.
  • Miễn phí cho một thiết bị
  • Giao diện sạch sẽ, dễ sử dụng
  • Tìm kiếm xuyên suốt tất cả file
  • Tự động đồng bộ với Cowork (cùng thư mục)

Tại sao Obsidian? Đáp ứng 3 tiêu chí: miễn phí, lưu trên máy tính, giao diện dễ dùng.

Quy trình hằng ngày

Obsidian: duyệt/tìm/chỉnh sửa → Cowork: xây dựng thành phẩm
Công cụVai trò
ObsidianCập nhật bộ não — duyệt, tìm, chỉnh sửa file context
CoworkXây dựng thành phẩm — đọc context từ thư mục, tạo output

Cấu trúc thư mục đề xuất

Thư mụcChứa gì
about-me/Thông tin cá nhân, phong cách viết, quy tắc
claude-output/Kết quả Claude tạo ra
templates/Mẫu ưa thích cho từng loại tài liệu
Mẹo: Trỏ cả Obsidian và Cowork vào cùng một thư mục. Chỉnh sửa trong Obsidian → Cowork tự đọc file mới nhất ngay lập tức.

Claude Skills (nâng cao)

Các quy trình đã lưu sẵn, gọi bằng lệnh /skill-name:

  • /newsletter — viết newsletter theo format chuẩn
  • /client-brief — tạo brief cho khách hàng
  • /sales-email — viết email bán hàng
Bước đầu tiên: dành 2 lần 20 phút/tuần để thiết lập cơ bản. Obsidian + Cowork + Skills = không bao giờ cần prompt từ đầu nữa.
