Huấn Luyện Claude Nói Giọng Của Bạn

Tạo file "giọng nói cá nhân" để AI viết đúng phong cách của bạn — portable trên mọi nền tảng

Nguồn: phandonggiang.com — Cấp độ 3, ~12 phút
Tư duy cốt lõi: Giọng văn, thẩm mỹ, cách chọn từ và tư duy của bạn có thể được nén vào một file text duy nhất. Bất kỳ AI nào cũng có thể "trở thành bạn" nếu có file này. Quy trình: phỏng vấn 100 câu hỏi → nén thành file markdown (~2.000-5.000 token).

Bước 1: Phỏng vấn 100 câu hỏi (trích xuất)

Chuẩn bị: Claude + Cowork, Model Opus, Extended Thinking bật, Wispr Flow (voice-to-text miễn phí).

Giao cho Claude vai trò "Taste Interviewer" — trích xuất DNA tư duy qua 7 danh mục:

Danh mụcSố câu
Niềm tin trái chiều & quan điểm15
Phong cách viết20
Những điều không thích (thẩm mỹ)15
Giọng điệu & cá tính15
Tổ chức nội dung15
Những điều tuyệt đối từ chối10
Dấu hiệu cảnh báo10
Quy tắc phỏng vấn: Hỏi từng câu một. Yêu cầu câu trả lời cụ thể + ví dụ. Chỉ ra mâu thuẫn. Đào sâu câu trả lời bất thường. Không chấp nhận "tôi thích sự đơn giản".

Thời gian: ~2 giờ, hoặc 90 phút nếu dùng speech-to-text.

Bước 2: Nén file (Voice Compiler)

Chuyển kho lưu trữ thô ~20.000 từ thành file .md nén gọn:

  • Độ dài mục tiêu: 2.000-4.000 token; giới hạn cứng 5.000 token
  • Bài test cốt lõi: "Nếu dòng này biến mất, AI có viết/chỉnh sửa/phán xét khác đi không?" → Giữ nếu có; cắt nếu không

File nén cần chứa

  • Quy tắc viết cụ thể
  • Từ chối cứng kèm ví dụ
  • Ví dụ tốt/xấu để so sánh
  • Cụm từ đặc trưng + từ ưa dùng/từ tránh
  • Cấu trúc câu đặc trưng
  • Thẩm mỹ rõ ràng
  • Quy tắc ra quyết định
  • 3-6 ví dụ gold-standard

Cấu trúc file About-Me

Hướng dẫn sử dụng → Voice fingerprint → Quy tắc viết → Từ chối cứng → Ví dụ

Voice fingerprint gồm: nhịp điệu, mật độ, sự trực tiếp, hài hước, nhiệt độ cảm xúc.

Thêm phần cuối: "Áp dụng profile trong im lặng trừ khi được override."

Kiểm tra thực tế

Mở phiên chat trống, trỏ thư mục Cowork vào file about-me. Claude đọc file trước mỗi response → output nghe y hệt giọng viết tự nhiên của bạn.

Mẹo: Test bằng cách cho Claude viết một đoạn ngắn về chủ đề bạn hay viết. So sánh với bài thật của bạn — nếu không nhận ra đâu là AI, file đã đạt chuẩn.

Portable — dùng mọi nơi

Khi có file about-me, bạn trở nên portable:

  • Dùng trên Claude, ChatGPT, Gemini, Grok, hoặc AI tương lai
  • Chia sẻ với ghostwriter hoặc team để viết theo giọng bạn khi vắng mặt
  • Bạn trở thành tài nguyên thay vì nút thắt cổ chai
Chỉnh sửa thường xuyên: Thẩm mỹ thay đổi theo thời gian. Dùng Obsidian (miễn phí) để mở và chỉnh sửa file .md dễ dàng — trỏ vào thư mục Cowork, tự đồng bộ.
Nắm bắt thẩm mỹ giúp tiết kiệm thời gian soạn thảo, giải phóng thời gian cho chỉnh sửa, tinh chỉnh, và xem xét lại cách tiếp cận.
