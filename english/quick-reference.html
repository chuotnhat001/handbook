Quick Reference

Tom tat cong thuc ngu phap — tra cuu nhanh

1. Sentence Structure

S + V + O (khang dinh)
S + do/does/did + not + V (phu dinh)
Do/Does/Did + S + V? (nghi van)

2. Present Simple

(+) I/You/We/They + V | He/She/It + V-s/es
(-) don't/doesn't + V
(?) Do/Does + S + V?

Use: habits, facts, schedules

3. Present Continuous

(+) S + am/is/are + V-ing
(-) S + am/is/are + not + V-ing
(?) Am/Is/Are + S + V-ing?

Use: now, temporary, plans

4. Past Simple

(+) S + V-ed / V2
(-) S + didn't + V
(?) Did + S + V?

Use: completed past actions

5. Past Continuous

(+) S + was/were + V-ing
(-) S + wasn't/weren't + V-ing
(?) Was/Were + S + V-ing?

Use: background action, when/while

6. Future

Will: S + will + V (decisions, predictions)
Going to: S + am/is/are + going to + V (plans, evidence)

7. Present Perfect

(+) S + have/has + V3
(-) S + haven't/hasn't + V3
(?) Have/Has + S + V3?

Use: experience, result still relevant, since/for

8. Modal Verbs

S + modal + V (bare infinitive, no "to", no -s)

can/could (ability) | should (advice) | must (obligation) | may/might (possibility)

9. Comparatives & Superlatives

Short adj: -er than / the -est
Long adj: more...than / the most...
Irregular: good-better-best, bad-worse-worst

10. Conditionals

Type 0: If + present, present (facts)
Type 1: If + present, will + V (real future)
Type 2: If + past, would + V (unreal)

11. Common Prepositions

Time: at (time), on (day), in (month/year)
Place: at (point), on (surface), in (space)
